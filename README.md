# @harborclient/team-hub-api

TypeScript client for the [HarborClient Team Hub](https://github.com/harborclient/team-hub) API.

## Install

```bash
pnpm add @harborclient/team-hub-api
```

## Development

```bash
pnpm install
pnpm lint
pnpm format:check
pnpm typecheck
pnpm test
pnpm build
```
